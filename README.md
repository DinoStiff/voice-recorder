# 🚕 Taipei AI Tour Guide (Hackathon Backend)

> 🏆 **專為黑客松打造的「Voice-First 智慧導覽大腦」！我們拋棄了傳統死板的下拉選單，完全突破地仰賴 AI 語音理解與情境感知，為旅客帶來如魔法般的沉浸式旅遊規劃體驗。**

## 🌟 核心破壞性創新 (Killer Features)

### 1. 🎤 純語音自然對話解析 (Voice-First / LLM Routing)
傳統旅遊 App 需要使用者狂點下拉選單，或靠勾選「戶外、文青」等 Emoji 標籤過濾資料。我們全面顛覆這個舊時代的產物！
- **動嘴不動手**：你只要對著麥克風說「我想要帶女朋友去浪漫的地方，但外面這陣子常下雨」。AI 大腦將自動拆解語音、濾除贅字，並精準翻譯為底層搜尋意圖。
- **無縫 LLM 路由**：這些口語化的意圖將被零延遲拋入大語言模型，由 AI 代替所有的 Click 動作，直接判定你真正渴求的行程脈絡。

### 2. 🗺️ Google Maps 即時環境打點 (Real-time Grounding)
徹底拋棄「死板」與「過時」的靜態假資料庫！
- **真實世界同步**：AI 拆解完需求後，會即時觸發過濾，並結合 Google Maps 搜刮「此時此刻真實存在、且正在營業」的店家進行打點。這確保推薦名單與現實 100% 同步，徹底消滅大語言模型常因缺乏時間感而產生的「AI 幻覺 (Hallucination)」。

### 3. 💕 情感感知與客群配對 (Vibe Match & Budget Actuary)
這裡的 AI 不只是景點推薦機器，它具備驚人的「財務心智」與「察言觀色」能力：
- **預算精算師 (Budget Actuary)**：當 AI 捕捉到總預算限制時，會聰明地進行資產分配。例如：自動扣除一項高單價的娛樂活動 (如：密室逃脫、看劇) 後，巧妙地利用剩下的預算去對接符合價位的隱藏版餐廳。
- **氛圍偵測 (Vibe Check)**：約會與遛小孩的語境截然不同！當偵測到「約會」詞彙，系統防火牆會直接封殺吵鬧的「親子遊樂園」，並大幅調升浪漫夜景、質感餐酒館的配對權重。

### 4. 🎴 直覺化配對介面 (Tinder-like Swipe UI)
誰說挑選行程只能看著一排枯燥的文字清單？我們賦予了系統最高黏著度的心流體驗：
- **左滑 / 右滑選景點**：像玩交友軟體一樣直覺且狂熱！對有興趣的景點卡片「右滑 (Like)」，不喜歡的「左滑 (Pass)」。用最低精神負擔的操作體驗，完美提升了系統的好感度與流暢度。

### 5. 🥊 1v1 終極二選一擂台賽 (Elimination Tournament)
選擇障礙者的超級福音！
- 萬一在 Tinder 卡片海中對太多地點都按了「喜歡」該怎麼辦？一旦地點超載，系統將無縫強制切換進入「**二選一淘汰擂台賽**」。每一回合殘酷的盲選對決，逼迫你做出破釜沉舟的取捨，直到最終誕生出最完美的一日遊行程時間軸！

### 6. 📍 GPS 座標準心與安心旅宿防護網
- **定位導航**：結合 GPS 動態掃描使用者半徑範圍的住宿地標。
- **合法登記認證**：完美對接政府開源的「合法旅宿靜態庫」，徹底排除高風險的法外日租套房，保護旅客身家。
- **C/P 值霸主比對系統**：爬蟲動態運算該地標的即時房價與歷史千百則回饋，進行幾何交叉運算，主動將「最高性價比 (CP 值)」的選項置頂！

## 📡 系統全新核心 API 路由 (API Endpoints)

### `POST /api/play_taipei/query` 
最新強化的 Gemini 大腦中樞節點。
徹底取代被淘汰的靜態查找機制，本路由直接接收「自然語言/語音辨識文本」，從這個單一入口引爆背後的 Vibe Match 演算法、Google Real-time Grounding，以及所有的卡片 Swipe 邏輯鏈！
